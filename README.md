# Connect4-Final-Assembly-Project
פרויקט גמר באסמבלי סוף כיתה י' שחף חן

על מנת להפעיל את הפרויקט צריך להוריד את כלל הקבצים למחשב לתיקיה הנבחרת, ולאחר מכן להריץ בעזרתDosbox

script for Dosbox-options.txt:
  mount c: c:\
  c:
  cd tasm
  cd bin

  tasm /zi project.asm
  tlink /v project.obj
  project.exe
